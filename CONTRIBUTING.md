# 贡献指南

感谢你愿意提交友情链接！

## 提交方式（推荐）

1. Fork 本仓库
2. 编辑 `data/friends.json`，按现有结构新增一个对象
3. 提交 Pull Request（PR）
4. 等待维护者审核与合并

## 字段说明

- `id`: 唯一标识，建议使用域名（例如 `example.com`）
- `name`: 站点名称（必填）
- `nameEn`: 英文名称（可选）
- `url`: 站点地址，必须是 `https://`（必填）
- `avatar`: 头像地址，建议为可长期访问的 `https://` 图片链接（必填）
- `sloganZh`: 中文简介（可选）
- `sloganEn`: 英文简介（可选）
- `createdAt`: 提交日期，格式 `YYYY-MM-DD`

## 收录建议

- 站点内容长期可访问
- 优先收录原创内容站点
- 请确保链接与头像可正常访问

## 注意事项

- 请不要提交违法、侵权或不适宜内容
- 维护者有权根据实际情况调整或拒绝收录
